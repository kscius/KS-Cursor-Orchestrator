# VS Code / Cursor extensions

Editor extensions (ESLint, GitLens, themes, etc.) **are not** installed by copying folders from this repository. Manage them with:

- **Cursor / VS Code → Extensions** (marketplace or VSIX).

This bundle focuses on **rules, commands, agents, skills, and hooks**. If your team depends on specific extensions, document the list in your internal wiki or add an optional `extensions/RECOMMENDED.md` here with extension IDs (`publisher.name`).
